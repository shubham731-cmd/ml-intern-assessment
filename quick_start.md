# Quick Start Checklist 

This checklist will help you get up and running with the N-Gram Language Model assignment in 10 minutes.

- [ ] **1. Fork the repository:**
  ```
  Click the "Fork" button (top right)
  ```

- [ ] **2. Clone the repository:**
  ```
  git clone <repo_link>
  ```

- [ ] **3. Install dependencies:**
  ```
  pip install -r requirements.txt
  ```

- [ ] **4. Implement the `TrigramModel` :**
Open trigram-assignment/src/ngram_model.py and fill in the TODO sections.

- [ ] **5. Run the tests:**
  ```
  pytest trigram-assignment/tests/test_ngram.py  

  Feel free to create your own tests.
  ```

- [ ] **6. Write your evaluation:**
  - Open `evaluation.md` and write a 1-page summary of your design choices.

- [ ] **7. Submit your assignment:**
  - Push your changes to your forked repository and create a pull request.
